# 🏛️📚 Library Management System (Arabic & English)

A web-based Library Management System with **dual-language support**, **dark mode**, **dynamic statistics**, and full management of books, members, borrowings, and reports.

---

## ✨ Features

- **Multi-Language:** Switch between Arabic & English dynamically.  
- **Dark Mode:** One-click toggle with seamless theme switching.  
- **Books & Members Management:** Add, edit, delete, and search dynamically.  
- **Borrowing System:** Borrow, return, and extend books.  
- **Dashboard & Statistics:** Live overview of library activity.  
- **Settings & Backup:** Customize library info and export/import data.  
- **Responsive Design:** Works on desktop, tablet, and mobile.

## 📦 Installation

### 1. Download the Project

Clone the repository:

```bash
git clone https://github.com/Alhomaidi1/Library-Management-System.git
````

Or download it as a ZIP and extract it.

---

### 2. Open in Browser

1. Navigate to the project folder
2. Open `index.html` in your preferred browser
3. No backend required; all data is stored in `localStorage`

---

## 🚀 Usage

1. Toggle language (Arabic/English) and theme (Light/Dark)
2. Navigate between pages:

   * Dashboard
   * Books
   * Members
   * Borrowings
   * Reports
   * Settings
3. Manage books, members, and borrowings using forms and tables
4. Export/Import data for backups
